# 텐서 클래스 아키텍처

## 목적

모델과 레이어들의 기초 밑바탕이 되는 텐서를 정의한다.

## 필요한 멤버 변수

- 텐서 관련된 메타데이터(차원 개수, 각 차원 길이, 현재 데이터 위치(gpu or cpu), Stride(각 차원에서 다음 요소까지의 거리) 등등)
- gpu, cpu 포인터
- 이 텐서와 연관된 연산 개체
- 이 텐서의 미분텐서

## 필요한 메서드

- 텐서 생성자
- gpu로 이동
- cpu로 이동
- 순전파 연산
- 역전파 연산
- 연산자 오버로딩
  - 원소별 덧셈
  - 원소별 곱셈
  - 행렬곱
  - 스칼라배
- 전치
